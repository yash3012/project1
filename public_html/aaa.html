AAA Testing

Importance of Testing

Software testing is the process of evaluating and verifying that a software product or application does what it is supposed to do. The benefits of testing include preventing bugs, reducing development costs and improving performance.

AAA Testing

This is a very important and fundamental idea of unit testing. In the overall view of any unit test application, a unit test is a three-step processes. The first one is setup test or arrange test, Act test or execute unit test and Assert means verify test result with expected results. The concept of three "A"s are given below.

Arrange:

This is the first step of a unit test application. Here we will arrange the test, We will do the necessary setup of the test. To perform the test we need to create an object of the targeted class, if necessary, then we need to create mock objects and other variable initialization.

Act:

This is the middle step of a unit step application. In this step we will execute the test. We will do the actual unit testing and the result will be obtained from the test application.

Assert:

This is the last step of a unit test application. In this step we will check and verify the returned result with expected results.

Example of AAA Testing in Python

What is AAA Testing?
Arrange-Act-Assert is a great way to structure test cases. It prescribes an order of operations: Arrange inputs and targets.
What are the three steps of AAA?
AAA has three basic steps:
  • Arrange
  • Act
  • Assert
